# Dota-2-Settler
Modify dota 2's settings with hotkeys

# Intial Release Hotkeys
Team
  Mute: ```-```
  Unmute: ```=```
Enemy
  Mute: ```[```
  Unmute: ```]```
Everyone
  Mute: ```;```
  Unmute: ```'```

# Dependencies
pynput
pyautogui
opencv-python

to install, simple run this command (assuming you have python3 installed)
```pip3 install pynput pyautogui opencv-python```
